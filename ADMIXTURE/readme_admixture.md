# ADMIXTURE Analysis — *Swiftia exserta* Population Genomics

Population structure analysis using ADMIXTURE for three datasets derived from clone-corrected, neutral SNP data (*Swiftia exserta*, northern Gulf of Mexico and Atlantic).

---

## Datasets

| Dataset | Individuals | Populations | SNPs | Description |
|---------|-------------|-------------|------|-------------|
| **Sub** (Gulf-Atlantic) | 174 | 10 (GoM + Edisto) | ~18,616 | Full dataset including Atlantic outgroup |
| **GoM10** | 158 | 9 (GoM only) | ~16,609–18,616 | Gulf of Mexico populations only |
| **GoMa** | 176 | 9 (GoM only) | ~18,616 | GoM with non-clonal singletons re-added |

All datasets are clone-corrected and filtered for neutrality (Ho filter → HWE filter → BayeScan outlier removal).

## Workflow

### Step 1 — Data preparation (R)
Run `scripts/01_data_prep.R` to convert genind objects to PLINK binary format. Requires `.Rdata` input files (not included; see Methods).

Output per dataset:
- `swiftia_<dataset>.ped` / `.map` → converted to `.bed` / `.bim` / `.fam` via PLINK

```bash
plink --file swiftia_GoM10 --make-bed --out swiftia_GoM10 --allow-extra-chr
```

### Step 2 — Run ADMIXTURE (shell)
Each script runs K = 1–10 with 10 replicates per K (fixed seeds for reproducibility), using 100 threads and cross-validation (`--cv`).

```bash
mamba activate popgen
chmod +x scripts/run_admixture.sh
nohup sh scripts/run_admixture.sh > admixture_GoM10_run.log 2>&1 &
```

Output directory structure: `admixture_results_<dataset>/K{k}/rep{rep}/log{k}_{rep}.out`

### Step 3 — Process results (R)
Run `R/02_process_results.R` to:
- Parse CV errors across all replicates and plot mean ± range by K
- Identify optimal K (lowest mean CV error)
- Generate faceted structure bar plots ordered west → east by longitude

## Notes

- K column assignments are arbitrary across ADMIXTURE runs (label switching). Identify the Gulf vs. Atlantic cluster by inspecting mean ancestry proportions in Edisto individuals before plotting.
- Population plotting order (west → east): WFGB → EFGB → Bright → Geyer → Elvers → McGrail → Bouma → Alderdice → Diaphus → FTR → AAR → RTR → Edisto
- Edisto is an Atlantic outgroup
- Colors used in structure plots: `#3491A8FF` (teal), `#C3E9CEFF` (light green), `#35264CFF` (deep purple)
